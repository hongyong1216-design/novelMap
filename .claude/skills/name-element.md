# 元墟元素命名技能

为小说《元墟》世界的稀有元素/天然矿物取名。两套并行：

- **本地名**：矿师/炼者凭肉眼手感命名，编码元素的**物理本相**（颜色·质地·形态），**与元粒子、三品灵性完全无关**——这是元墟本地的「炼物学」传统，如同地球古人以偏旁（钅石氵气）编码物质态。
- **外来者名**：星槎号地球人按科学**原子序数**编号。

> ⚠️ 分层铁律：**元素 = 星球的凡矿基础物质层**（元粒子养出的重元素本身，纯物理）；**元晶/元石 = 元素＋元粒子结合后的活矿**（已定名鸣金/磐金/流光/赤锉，见 `backgroundCards.js` item-1）。二者是上下两层，命名**严禁混用**：元素走 金/石/精/液 这套纯形态通名，不得占用「鸣/磐/流光/赤锉」四个活矿词。命名前先读 `story-1`（元墟卡·稀有元素成因）与 `item-1`（元晶）对齐。

## 世界观锚点（取名前先懂，稀有度由此而来）

1. **元粒子「养」出重元素**：越重越复杂的元素，元粒子培育越久 → 越稀有 = 埋在越古老地层/越深地脉。稀有度对应「纪元深度」与「产地深度」，而非粒子含量。
2. **锁合稳住「不该存在的元素」**：元粒子锁合之力能稳定别处会剧烈衰变的超重元素（对标真实「稳定岛」理论）。元墟最稀者＝全宇宙独此一份的稳定超重元素，恰好扎堆在稳定岛幻数 114/120/126……——地球人测出时即一记「世界观击穿」。
3. **离脉即衰**：越太古的元素越依赖粒子锁合，一旦离开富粒子地脉环境即失稳、衰变、暴走。稀有度越高＝越危险、越难保存。

## 一、本地名：颜色 ＋ 质地 ＋ 类属

```
本地名 =（颜色字）+（质地字）+（类属·通名字）
```
字数 3 字为主。稀有度**不靠粒子**，靠可选的「产地/来路」冠字体现。

### 第①段 · 颜色（首字，最先入眼）

| 色系 | 可选字 |
|---|---|
| 红 | 赤・丹・绛・彤 |
| 黄 | 缃(浅黄)・缇(橙黄) |
| 青绿 | 苍・绀(深青)・碧・黛 |
| 黑 | 玄・墨・乌 |
| 白 | 缟・皎・素 |
| 紫/银灰 | 紫・银・铅 |

### 第②段 · 质地（中字，物理性状）

| 维度 | 可选字 |
|---|---|
| 轻重 | 重・沉・轻 |
| 坚脆 | 坚・刚・韧・脆・绵 |
| 冷热感 | 寒・温・灼 |
| 光泽 | 曜(有光)・莹・晦(黯) |

### 第③段 · 类属通名（尾字，回答「是什么形态的东西」，纯物理与灵性无关）

| 类属 | 通名字 | 含义 |
|---|---|---|
| 金属 | 金・铁 | 有金属光泽、可锻 |
| 矿石 | 石・砂・砾 | 非金属固体，成块或颗粒 |
| 结晶 | 精・晶・英 | 天然结晶、透莹（「晶」慎用，防混元晶） |
| 液态 | 液・膏・汞 | 常温流动或半流 |
| 气态 | 气・霭 | 常温挥发 |
| 升华 | 华・霜 | 易升华成霜 |

### 可选变体

**A. 第②段可换「性」段**——当元素的天然脾性比质地更抢眼时，用反应特征命名（更写实，如地球「氯＝绿气」）：

| 脾性 | 字 | 例 |
|---|---|---|
| 遇火燃 | 燃・焚 | 赤焚砂 |
| 遇水变 | 沁・沃 | 苍沁石 |
| 腐蚀/毒 | 蚀・毒 | 玄蚀液 |
| 磁性 | 磁・引 | 乌磁铁 |
| 鸣响 | 鸣 | 铮鸣石 |
| 易变色/态 | 变・幻 | 幻曜精 |

**B. 稀有度靠「来路」冠首**（只在强调稀有/来历时加，日常可省）：

| 来路 | 冠字 | 例 |
|---|---|---|
| 太古地层 | 太・古 | 太玄重金 |
| 极深地脉 | 幽・渊 | 幽苍寒液 |
| 陨落天外 | 陨・天 | 陨缟莹精 |

## 二、外来者名：X ＋ 编号

沿用元晶外来者法理（未知物冠 X），但元素有原子序数可测，故编号从「字母级」升级为「数字序号」，更像科学编目。

```
外来者名 = X +（编号）+（可选特征后缀）
```

- **地球人认得的元素**（已知周期表内）→ 不冠 X，直接用原名与符号（铁 Fe、铀 U……）。
- **元墟独有元素**（排不进已知周期表的稳定超重元素）→ **X ＋ 推测原子序数**：如 `X-114`、`X-120`、`X-126`。序数越大＝越重＝越稀有＝越近稳定岛幻数。测不准时用 `X-?` 或区间 `X-11x`。
- **可选特征后缀**：序数难测或需口语区分时，缀地球人观察到的现象词，如 `X-126「Umbra」`（拉丁「暗影」，因其玄黑）、`X-120「Ashfall」`（离脉即衰如落灰）。
- 地球人私下统称这批矿为「岛素」（稳定岛元素）。

## 使用方式

```
/name-element [要命名的元素描述]
```
描述尽量给出：**颜色 / 质地或脾性 / 形态(金属?矿石?液?) / 稀有度或产地 / 推测原子序数**，越具体产出越准。

示例：`/name-element 黑色、极重、金属、太古深地所出、推测序数126`

## 输出格式

对每个待命名元素，产出一行对照：

| 本地名 | 拆解（颜色＋质地＋类属＋可选冠字） | 外来者名 | 一句本相/设定 |
|---|---|---|---|

并附一句说明——好的元墟元素名，**本地人凭感官命名（这是会暴走的太玄重金），地球人凭测量命名（这是稳定岛的 X-126），同一块矿两套认知，正是世界观碰撞的缩影**。

## 避坑

1. **与元晶彻底分层**：鸣金/磐金/流光/赤锉是含粒子的活矿元晶；元素名走纯形态通名，勿混。
2. **「金」可作元素金属通名**（单字），但避开已占的「鸣/磐/流光/赤锉」。
3. **「精/晶」慎用**：可作结晶类通名，但语境须清楚，别和元晶的「晶」混。
4. **本地名一元素一正名**：三段定了就是户口，不随便改字。
5. **本地名不编码灵性/粒子**：稀有度、危险性只由「来路冠字」或元素本身脾性体现，绝不出现三品(凡/灵/神)或粒子含量字眼——那是元晶层的事。
