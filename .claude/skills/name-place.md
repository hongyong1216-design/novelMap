# 元墟地名生成技能

为小说《元墟》世界的各类地点批量取名。核心思想：**元墟的地名一名双编码——物理地理定「骨」（山川气候），灵能地理定「魂」（灵潮/粒子态/意义场/纪元）**。一个好地名，一听就知道那地方能量如何、意义如何、属哪个纪元。

> 命名前若涉及具体设定，先读 `src/pages/NovelEditorPage/subpages/StoryBackground/data/backgroundCards.js`（尤其 `story-1` 元墟卡）对齐三日、气候带、元系物产等术语，勿造玄虚新词。

## 总则

**物理地理定「骨」，灵能地理定「魂」。** 一个地名 = 它长什么样 ＋ 它的能量/意义/纪元是什么。通常一「骨」一「魂」配一个通名。

## 基本公式

```
地名   =（取意字 1~2）+（地类通名）
大地名 =（方位/序数）+（取意字）+（通名）    ← 世界级大区用
```
字数控制在 **2~4 字**。

## 通名表（决定「多大、什么类的地方」）

| 尺度（地图层级） | 通名 | 用于 |
|---|---|---|
| **世界级 L0**（中性辽阔） | 洲・域・原・陆・土・野・甸・垓 | 大陆、势力总疆 |
| **世界级 L0**（险绝死域） | 荒・漠・绝・墟※ | 炽盛/震荡带、幽寒荒原 |
| **世界级 L0**（神性天界） | 天・界・穹・垣・阙 | 圣地、三日辐照核心、古文明中枢 |
| **世界级 L0**（水泽海洋） | 溟・海・瀚・泽・渊 | 海域、大泽、背光冰洋 |
| **区域级 L1** | 境・野・川・林・泽・海 | 一片地区、地形区 |
| **地点级 L2** | 城・关・邑・寨・港・坊・驿 | 聚落、要冲 |
| **山地** | 岭・峰・嶂・崖・岫 | — |
| **水系** | 川・河・泽・渊・溟・津・涧 | — |
| **险地** | 峡・隘・堑・壑・陉 | — |
| **遗迹/墟场** | 墟・冢・垒・垣・殿・窟・迹 | 文明地层、古战场 |
| **灵能/矿脉** | 脉・眼・渊・窟・祠 | 地脉、元晶矿、圣地 |

※「墟」是星球本名（元墟）之字——**克制使用，只留给最古老、最接近文明地层核心的世界级遗迹**（如封印机心的太古之地）。一用即有「直通太初」的分量，滥用则失重。

| 维度 | 取意方向 | 可选字 |
|---|---|---|

| | 背光幽寒 | 幽・玄・晦・冥・朔・寒・烛 |
| | 无常震荡 | 荡・澜・汐・涌・变 |
| | 恒稳宜居 | 宁・恒・安・稳・和 |
| **② 粒子态** | 活跃（灵潮盛） | 灵・鸣・活・旺・涌 |
| | 沉睡 | 眠・寂・蛰・沉・寐 |
| | 饥饿/死域 | 枯・槁・竭・朽・绝 |
| | 过载/暴走 | 暴・狂・溃・焚・爆 |

| | 悲（悲氛） | 悲・哀・怨・殇・泣 |

| **⑥ 纪元·地层** | 太古/前纪元遗存 | 太・邃・古・遗・先・旧 |

**抽法**：从 ①~⑥ 里挑 1~2 个维度的字（通常一「骨」一「魂」），配一个通名。

## 进阶：一地三名（重要地点用，沿用元晶的「三套名」传统）

- **本地雅名**（诗意，主用）：如 `眠雪原`
- **外来者名**（地球人按现象直呼，沿用元系 X+特征命名法）：如 `X-死寂雪区`
- **神话俗称**（民间口传）：如 `龙背之地`

## 使用方式

```
/name-place [要命名的地方描述]
```
描述里尽量给出：**气候带 / 地貌 / 灵能性质 / 所属势力 / 纪元**，越具体产出越准。

示例：`/name-place 炽盛带边缘、鸣金大矿脉、某势力的采矿重镇`

## 输出格式

对每个待命名地点，产出一张小表：

| 主名（雅名） | 拆解（骨+魂+通名） | 外来者名 | 神话俗称 |
|---|---|---|---|

并附一句「为什么这个名字在暗示什么」——好的元墟地名，**名字本身在剧透，却只对读懂设定的人剧透**。

## 范例

| 待命名 | 主名 | 拆解 |
|---|---|---|
| 温稳带·王城核心 | **恒昭城** / 雅名`沃宁原` | 恒稳 + 太昭主日 + 城 |
| 幽寒荒原深处 | **邃冥荒** / 俗称`死月之下` | 邃曜(南日)微光 + 幽冥 + 荒 |
| 一处古战场 | **煞戾墟** / **血殇冢** | 煞氛 + 遗迹通名 |
| 炽盛带能量渊 | **焱曜渊** | 阳盛过曝 + 水系通名 |
| 鸣金大矿脉 | **鸣雷脉** | 元系物产 + 矿脉通名 |
| 前纪元机心封印处 | **邃蛰垒** | 太古 + 沉睡（蛰，呼应封印的机心）+ 遗迹 |
| 圣地神木林 | **虔灵林** / 雅名`圣木境` | 圣氛 + 活跃粒子 + 林 |

`邃蛰垒` 一名即暗示「太古 + 有物沉睡 + 是座遗迹」——正扣封印的征服机心。这就是双编码：名字在剧透，只对懂的人剧透。
